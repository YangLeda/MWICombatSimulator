【MWITools 插件】

主要更新地址：https://greasyfork.org/en/scripts/494467-mwitools

备用（需要科学网络）：https://raw.githubusercontent.com/YangLeda/Userscripts-For-MilkyWayIdle/main/MWITools.js


包含许多种功能，一般在游戏中以绿字显示。


设置里可以开关各种脚本功能

市场价格数据来自 https://github.com/holychikenz/MWIApi，我不清楚是否为官方发布

更新市场价格数据需要科学网络，否则使用的是旧数据。如果你网络问题无法更新，一般会在游戏网页右上角显示红字。

市场价格数据是24小时均价，大约每半小时更新一次。

价格 "/" 前后数字，前者是卖单价，后者是买单价。

带强化等级的装备没有市场价格数据，插件可能以价值0金币处理，可能改为按强化模拟处理。

如果有问题，关闭其它插件试试，可能有冲突。

iOS手机Safari安装插件方法：应用商店搜索安装Stay，并按照Stay的提示进行初始设置；通过备用链接下载脚本；脚本设置：运行时机选document-start、注入方式选Page。

不喜欢绿色字体的话，可以在代码开头处自行修改，代码开头处有相关提示。

游戏并没有直接提供“这场战斗中消耗了哪些消耗品”这种信息，插件难以准确地统计，因此插件的战斗总结中只有经验和收入，没有消耗品支出。没有做此统计的计划。


网络问题排查：iOS手机似乎不支持插件访问网络，无解。各种手机浏览器也可能有不同的限制/设置/bug，我无法定位问题。

网络问题排查：如果你用浏览器无法稳定在5秒钟以内打开 https://raw.githubusercontent.com/holychikenz/MWIApi/main/medianmarket.json ，那么一定是你网络问题，请自行想办法解决。

网络问题排查：请更换为新版本的 Chrome / Edge / Firefox 浏览器，油猴插件也更新到新版。如果你使用很老版本的浏览器或油猴插件，或者使用非世界主流的浏览器或油猴插件，或者你的公司电脑做了某种限制，那么我无法定位问题。

网络问题排查：如果以上几点都排除了，仍需要定位问题，请打开浏览器控制台，刷新游戏网页，截图发给我。

目前已启用市场API备用地址：http://43.129.194.214:5500/apijson，github获取失败会尝试备用地址（香港服务器）。不保证长期维护。

【战斗模拟（批量） 网站】

网址：http://43.129.194.214:5000/mwisim.github.io


一键模拟所有战斗地图，以表格形式比较各地图经验和利润。


基于mwisim战斗模拟网站修改。

此网站维护需要手动同步原版模拟网站的版本更新，且需要服务器购买成本，不保证长期存在。

如果你在导入数据的时候弹窗"structuredClone()"之类的报错，说明你的浏览器版本过旧，请使用较新版本的 Chrome/Edge/Firefox 等浏览器。此问题来自原版mwisim战斗模拟网站。

人物信息全都空白直接模拟会报错。刷新网页，导入或者填写合理的人物信息再模拟。此问题来自原版mwisim战斗模拟网站。
